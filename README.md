# FizzBuzz
Description: Iterate i through 1 to N. 
            If i is divisible by 3, add Fizz 
            If i is divisible by 5, add Buzz
            if i is divisible by 7, add Fuzz
            ...
            if i is divisible by k, add STRING
Scalable solution.
